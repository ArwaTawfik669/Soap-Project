VVDCalculatorSoap

SOAP Version: 1.1

Main class which you can use to connect to your web service

Important settings

  • Timeout - the timeout for network operations, in milliseconds. By default: 60 sec.
    Example:
                        
VVDCalculatorSoap service = new VVDCalculatorSoap(null,"http://www.dneonline.com/calculator.asmx",30000);
service.Add();
  • Url - the web service address. By default it is retrieved from the wsdl file
    Example:
                        
VVDCalculatorSoap service = new VVDCalculatorSoap(null,"http://www.dneonline.com/calculator.asmx");
service.Add();
  • httpHeaders - allows you to add custom headers to the http requests.
    Example:
                        
VVDCalculatorSoap service = new VVDCalculatorSoap();
service.httpHeaders.add( new HeaderProperty("Authorization","Basic QWxhZGRpbjpPcGVuU2VzYW1l"));
service.Add();
  • Enable logging - if you enable logging, all requests/responses will be write to the console (for debugging purposes)
    Example:
                        
VVDCalculatorSoap service = new VVDCalculatorSoap();
service.enableLogging=true;
service.Add();
